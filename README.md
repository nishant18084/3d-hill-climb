# 🌌 RaceDrive - Movies & Games Galaxy Hub

यह एक कस्टमाइज्ड और सुपरफास्ट वेब एप्लीकेशन है जो वीडियो स्ट्रीमिंग और HTML5 गेम्स (जैसे Subway Surfers) को एक ही जगह पर डार्क-नेटफ्लिक्स लुक में प्रदान करता है।

## 📁 5 फाइलों का मुख्य आर्किटेक्चर (Project Structure):
1. `index.html` - मुख्य एंट्री पॉइंट और वेबसाइट का लेआउट।
2. `style.css` - स्टाइलिंग, कलर्स और सुंदर UI एनिमेशन।
3. `data.js` - मूवी लिंक्स और गेम्स गैलेक्सी का पूरा डेटा।
4. `controls.js` - वीडियो प्लेयर इंजन, फुलस्क्रीन और गेम पॉपअप कंट्रोल्स (Sandbox Security के साथ)।
5. `app.js` - मुख्य इनिशियलाइज़र जो वेबसाइट खुलते ही सब कुछ स्क्रीन पर लोड करता है।

## 🚀 लाइव सेटअप:
1. इन सभी फाइलों को गिटहब रिपोजिटरी में अपडेट/क्रिएट करें।
2. गिटहब ऐक्शन्स (Pages build and deployment) के पूरा होने के लिए 1 मिनट रुकें।
3. आपकी वेबसाइट कस्टमाइज्ड गेम पॉपअप और इन-बिल्ट सुरक्षा के साथ लाइव हो जाएगी!
