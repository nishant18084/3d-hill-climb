# 🌌 RaceDrive - Movies & Games Galaxy Hub

यह पूरी तरह से कस्टमाइज्ड और मॉड्यूलर (अलग-अलग फाइलों वाला) वेब ऐप है, जिसमें मूवी प्लेयर के साथ गेम्स गैलेक्सी भी मौजूद है।

## 📁 आर्किटेक्चर गाइड:
1. `index.html` - मुख्य द्वार
2. `style.css` - स्टाइलिंग और डार्क थीम डिज़ाइन
3. `movies-data.js` - मूवी लिंक्स
4. `games-data.js` - कस्टमाइज्ड गेम लिस्ट (Subway Surfers)
5. `player-core.js` - वीडियो प्लेयर लॉजिक
6. `games-core.js` - गेम पॉपअप लॉजिक
7. `ui-control.js` - फुलस्क्रीन और टैब नेविगेशन
8. `components.js` - लेआउट टेम्पलेट्स
9. `app.js` - मुख्य इनिशियलाइज़र मैनेजर
10. `README.md` - प्रोजेक्ट गाइड

## 🚀 लाइव कैसे देखें?
सभी फाइलों को रिपॉजिटरी में सेव करके 1 मिनट रुकें, वेबसाइट पूरी तरह लाइव काम करने लगेगी!
